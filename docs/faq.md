!!! question "Apakah aplikasi ini gratis?"

!!! question "Apakah bisa digunakan offline?"

!!! question "Bagaimana cara update aplikasi?"

    Gak perlu ribet! Aplikasi otomatis update sendiri pas kamu buka. Kalo ada update gede, bakal muncul notifikasi suruh refresh. Klik aja, selesai. Ez!

## :material-login: Login & Akun

!!! question "Lupa password, tapi tidak terima email reset?"

    Cek folder Spam/Junk dulu (99% nyasar kesana). Masih gak ada juga? Chat admin aja, minta direset manual. Gak usah malu, kita semua pernah kok!

!!! question "Kenapa tidak bisa login? Error terus!"

    **Coba cek ini dulu:**

    - Email sama password bener gak? (huruf besar/kecil ngaruh loh!)
    - Internet nyala? Coba buka Google dulu, kalo bisa berarti internet OK
    - Akun udah diaktifkan admin? Kalo baru daftar, tunggu konfirmasi dulu
    - Masih error? Clear cache browser atau coba browser lain

!!! question "Apakah harus verifikasi email? Ribet amat!"

    Iya harus. Ini buat keamanan akun kamu sendiri loh! Daripada nanti di-hack terus data pelanggan bocor, kan repot. 1 menit doang kok, gampang!

## :material-package-variant: Pesanan

!!! question "Kenapa tombol 'Jemput Pesanan' tidak muncul?"

    **Ada beberapa kemungkinan:**

    - Udah diambil kurir lain (siapa cepat dia dapat!)
    - Status pesanan bukan "Menunggu"
    - Refresh halaman dulu, mungkin belum update

!!! question "Bisa cancel pesanan yang sudah diambil gak?"

    Gak bisa via aplikasi. Kalo ada kendala (macet parah, ban bocor, hujan deras, dll), langsung hubungi admin. Mereka bakal bantu kok!

!!! question "Berapa lama data pesanan tersimpan?"

    **SELAMANYA!** Semua pesanan disimpan permanent. Jadi kalo mau liat pesanan dari tahun kemarin juga bisa. Berguna buat evaluasi kinerja kamu loh!

## :material-navigation: Navigasi & GPS

!!! question "GPS tidak akurat / loncat-loncat kayak kursor lemot?"

    Ah, klasik! GPS lagi galau nih. Coba solusi ini:

    1. **Pindah ke tempat terbuka** - GPS butuh "lihat" langit, bukan atap gedung
    2. **Restart aplikasi** - Kadang cuma perlu di-refresh doang
    3. **Cek pengaturan GPS** - Set ke mode "High Accuracy" atau "Presisi Tinggi"
    4. **Sabar dikit** - Tunggu 10-20 detik biar GPS "ngunci" satelit
    5. **Masih ngaco?** - Mungkin HP-nya yang bermasalah. Coba restart HP

!!! note "Real Talk"
    GPS di gedung bertingkat atau basement itu kayak main petak umpet sama satelit. Susah!

!!! question "Rute tidak muncul di peta? Kosong melompong!"

    **Kemungkinan besar:**

    - **Internet lelet** - Peta butuh download data, kalo internetnya 1 bar ya... yaudah
    - **Koordinat pelanggan belum di-set** - Adminnya lupa input. Chat aja minta di-update
    - **Bug sementara** - Refresh halaman atau restart aplikasi

!!! question "Apakah GPS tracking terus jalan meski app-nya di-minimize?"

    Gak kok! GPS cuma nyala pas kamu buka halaman "Rute Detail". Tutup halaman = tracking berhenti. Jadi aman, gak disadap terus 24/7

## :material-chat: Chat

!!! question "Pesan tidak terkirim? Stuck loading terus!"

    **Beberapa hal yang perlu dicek:**

    - **Internet mati?** - Ya iya lah gak bisa kirim
    - **File kegedean?** - Max 5 MB ya! Kalo lebih, compress dulu
    - **Format file aneh?** - Cuma support JPG, PNG, GIF, PDF, DOC, DOCX
    - **Server lagi maintenance?** - Jarang sih, tapi bisa aja. Tunggu 5-10 menit trus coba lagi

!!! question "Apakah chat tersimpan? Bisa diliat lagi gak?"

    Bisa! Semua chat tersimpan di server. Mau ngobrol dari bulan lalu juga masih ada. Kecuali kamu sendiri yang hapus percakapannya.

!!! question "Bisa chat dengan sesama kurir gak? Buat koordinasi gitu"

    Gak bisa. Chat cuma buat komunikasi sama Admin atau Pelanggan aja. Kalo mau chat sama kurir lain, pake WhatsApp grup kek atau apalah

## :material-cash: Pembayaran

!!! question "Upload bukti pembayaran untuk apa sih?"

    Buat mempercepat verifikasi aja! Admin bakal cek foto bukti bayar kamu, terus update status pembayaran. Jadi pelanggan happy, admin happy, kamu juga happy! Win-win-win!

!!! question "Apakah kurir yang terima uang dari pelanggan?"

    **Tergantung kebijakan laundry kamu:**

    - **Tunai:** Biasanya kurir yang terima, terus disetor ke admin/kasir
    - **Non-Tunai:** Pelanggan transfer langsung ke rekening laundry via QRIS/transfer bank

    Tanya admin kamu dulu biar jelas ya!

!!! question "QR Code yang ada di detail pesanan itu buat apa?"

    Itu QR Code QRIS buat pembayaran non-tunai. Pelanggan tinggal scan pake app mobile banking (GoPay, OVO, Dana, BCA Mobile, dll), langsung bayar. Praktis!

## :material-account-cog: Profile & Pengaturan

!!! question "Kenapa harus set lokasi rumah? Aman gak sih?"

    **Aman kok!** Ini buat:

    - Admin bisa liat kurir terdekat dari lokasi pelanggan (biar efisien)
    - Navigasi jadi lebih akurat
    - Koordinasi lebih gampang

    Lokasi kamu cuma keliatan sama admin, gak dipublish ke publik. Santai aja!

!!! question "Upload avatar gagal terus! Kesel deh!"

    **Coba cek ini:**

    - **Format file** - Harus JPG atau PNG (bukan GIF, BMP, atau format aneh lainnya)
    - **Ukuran** - Max 2 MB! Kalo lebih, compress dulu pake app (TinyPNG, Compressor.io, dll)
    - **Koneksi internet** - Pastikan stabil, jangan 1 bar doang

!!! tip "Tips"
    Foto avatar gak usah HD 4K segala. Yang penting keliatan muka, gak blur. Simple is the best!

!!! question "Mode gelap gak jalan? Udah di-toggle masih terang aja!"

    Coba refresh halaman dulu (F5 atau swipe down). Masih gak jalan juga? Clear cache browser. Terakhir, restart aplikasi. Pasti jalan deh!

## :material-tools: Teknis (Troubleshooting)

!!! question "Aplikasi lambat banget! Kayak lagi loading jaman dial-up!"

    Wah, nostalgia jaman modem ya. Coba ini:

    1. **Clear cache browser** - Kayak bersihin sampah, perlu rutin
    2. **Tutup tab browser lain** - RAM HP juga terbatas, kasian dia
    3. **Restart aplikasi** - Kadang cuma butuh istirahat sebentar
    4. **Reinstall PWA** - Uninstall dari home screen, install ulang
    5. **Terakhir:** Cek HP kamu, storage penuh gak? RAM cukup gak?

!!! question "Error terus saat upload foto! Frustasi deh!"

    **Coba langkah ini:**

    1. **Compress foto** - Pake app compressor (gratis banyak kok)
    2. **Pake format JPG** - Format paling aman, jarang error
    3. **Cek internet** - Jangan upload pas sinyal cuma 1 bar
    4. **Upload satu-satu** - Jangan sekaligus banyak, biar gak nge-lag

!!! question "Halaman blank / gak muat apa-apa! Putih doang!"

    Refresh halaman (F5 atau swipe down). Masih blank? Clear cache browser. Masih juga? Restart HP. Kalo masih tetep blank, chat admin. Mungkin ada maintenance server
