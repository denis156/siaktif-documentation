# :material-cog: Pengaturan

Halaman untuk mengatur aplikasi dan profil Anda.

![Screenshot Pengaturan](../images/menu-utama/screenshot-pengaturan.png)

---

## :material-palette: Tema

<div class="grid cards" markdown>

-   :material-theme-light-dark:{ .lg .middle } **Mode Gelap**

    ---

    Toggle untuk mengaktifkan/menonaktifkan dark mode

    Mengurangi ketegangan mata dan menghemat baterai

</div>

!!! tip "Hemat Baterai"
    Mode gelap sangat membantu menghemat baterai terutama pada layar OLED/AMOLED!

---

## :material-account: Profil

<div class="grid cards" markdown>

-   :material-account-edit:{ .lg .middle } **Kelola Profil**

    ---

    Link ke halaman edit profil lengkap

    Perbarui nama, nomor HP, email, dan alamat

</div>

!!! info "Update Profil"
    Pastikan data profil Anda selalu up-to-date untuk memudahkan komunikasi dan koordinasi

---

## :material-security: Keamanan

<div class="grid cards" markdown>

-   :material-lock-reset:{ .lg .middle } **Ubah Password**

    ---

    Ganti password akun Anda secara berkala untuk keamanan

-   :material-logout:{ .lg .middle } **Keluar dari Sistem**

    ---

    Logout dari aplikasi dan kembali ke halaman login

</div>

!!! warning "Keamanan Akun"
    Disarankan untuk mengubah password secara berkala (setiap 3-6 bulan) untuk menjaga keamanan akun Anda
