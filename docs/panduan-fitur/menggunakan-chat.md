# :material-message-text: Cara Menggunakan Chat

Panduan berkomunikasi dengan Admin dan Pelanggan.

![Screenshot Chat Room](../images/chat/screenshot-chat-room.png)

---

## :material-open-in-app: Membuka Chat

<div class="grid cards" markdown>

-   :material-bell:{ .lg .middle } **Dari Notifikasi**

    ---

    - Klik badge notifikasi di **ikon chat** (top nav)
    - Pilih percakapan

-   :material-chat:{ .lg .middle } **Dari Menu Chat**

    ---

    1. Buka menu **"Chat"** dari top navigation
    2. Cari atau klik percakapan yang ingin dibuka

</div>

---

## :material-message-text-outline: Mengirim Pesan Text

!!! info "Cara Mengirim Pesan"

    1. Ketik pesan di kolom input bawah
    2. Klik tombol **"Kirim"** atau tekan Enter
    3. Pesan akan muncul di sisi kanan (bubble biru)
    4. Status "Terkirim" atau "Dibaca" akan muncul di bawah bubble

---

## :material-file-image: Mengirim File/Foto

!!! tip "Langkah Upload File"

    1. Klik ikon **Paperclip** di sebelah kiri kolom input
    2. Pilih file dari galeri atau ambil foto baru
    3. Modal preview akan muncul
    4. (Opsional) Tulis pesan untuk file tersebut
    5. Klik **"Kirim"**

**Format yang didukung:**

<div class="grid cards" markdown>

-   :material-image:{ .lg .middle } **Gambar**

    ---

    JPG, JPEG, PNG, GIF

-   :material-file-document:{ .lg .middle } **Dokumen**

    ---

    PDF, DOC, DOCX

</div>

!!! warning "Ukuran File"
    **Ukuran maksimal:** 5 MB per file

---

## :material-eye: Melihat File yang Dikirim

<div class="grid cards" markdown>

-   :material-image-outline:{ .lg .middle } **Untuk Gambar**

    ---

    - Gambar akan tampil inline di bubble chat
    - Klik gambar untuk lihat fullscreen
    - Klik tombol Download untuk simpan

-   :material-file-document-outline:{ .lg .middle } **Untuk Dokumen**

    ---

    - Nama file akan tampil dengan ikon paperclip
    - Klik untuk download dan buka

</div>

---

## :material-message-plus: Membuat Chat Baru

!!! success "Langkah-langkah"

    1. Klik tombol **"Buat Chat"** di halaman Chat
    2. Pilih **Tipe Participant:**
       - **Admin** - Untuk chat dengan admin/kasir
       - **Pelanggan** - Untuk chat dengan pelanggan
    3. Pilih **Nama Participant** dari dropdown
    4. Klik **"Buat Chat"**
    5. Chat room akan otomatis terbuka

---

## :material-delete: Menghapus Percakapan

!!! warning "Perhatian: Tindakan Permanen"

    1. Buka chat room yang ingin dihapus
    2. Klik tombol **"Hapus"** (merah) di header
    3. Modal konfirmasi akan muncul
    4. Klik **"Ya, Hapus"**
    5. Percakapan dan semua file akan terhapus permanen

!!! danger "Peringatan"
    File yang dihapus tidak bisa dikembalikan!

---

## :material-auto-fix: Fitur Otomatis

<div class="grid cards" markdown>

-   :material-arrow-down:{ .lg .middle } **Auto-scroll**

    ---

    Otomatis scroll ke pesan terbaru saat ada pesan baru

-   :material-refresh:{ .lg .middle } **Auto-refresh**

    ---

    Pesan baru otomatis muncul tanpa refresh manual

-   :material-check-all:{ .lg .middle } **Mark as Read**

    ---

    Pesan otomatis ter-mark "Dibaca" saat dibuka

</div>
